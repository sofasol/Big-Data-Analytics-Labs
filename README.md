# Анализ больших данных (Big Data Analysis)

Репозиторий для лабораторных работ по курсу "Анализ больших данных".

## Структура репозитория

| Папка | Лабораторная работа | Описание |
|-------|---------------------|----------|
| `Lab1_Snowflake` | №1 | Нормализация данных в схему "снежинка" |
| `Lab2_Spark` | №2 | Обработка данных с использованием Apache Spark |
| `Lab3_Flink` | №3 | Потоковая обработка данных с Apache Flink |

## Стек технологий

- PostgreSQL
- Docker
- Apache Spark
- Apache Flink
- SQL

## Запуск лабораторных работ

Каждая лабораторная работа содержит собственные инструкции по запуску и выполнению.
